# 🧠 Web Development Quiz App

A simple and interactive **Web Development Quiz Application** built using **HTML, CSS, and JavaScript**. This project allows users to test their knowledge with multiple-choice questions and get instant results.

---

## 🚀 Features

- ❓ 10 Multiple Choice Questions
- ⚡ Instant Answer Feedback
- 📊 Score Display at the End
- 🔄 Restart Quiz Option
- 🎨 Clean and User-Friendly UI
- 📱 Fully Responsive Design

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the app  
- **CSS3** – Styling and layout  
- **JavaScript (Vanilla JS)** – Quiz logic and interactivity
